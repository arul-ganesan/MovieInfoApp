# Movies Info App Using ASP.NET Core and Angular

This is a code repository for a MoviesInfo application (originally created as part of a developer challenge for an interview).

The application/database/api has been deployed to Azure and the website can be accessed at https://moviesinfoapp.azurewebsites.net/

## Requirements/Spec:

Allows users to,

	1. Search for a Title by its name.
	2. Display the titles that match the above criteria.
	3. Allow the user to pull up detailed information about a selected title from the list.

## Technology stack:

	1. Bootstrap5
	2. Angular13
	3. WebAPI
	4. Entity Framework

## Data:

SQL Server Express

Restore “Titles.bak” from the Database Backup folder using the instructions from the link below.
